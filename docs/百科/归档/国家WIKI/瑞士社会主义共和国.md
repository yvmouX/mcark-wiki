# 瑞士社会主义共和国<!-- {docsify-ignore-all} -->

###### 多元社会主义和平国家

​    瑞士社会主义共和国(Swiss Socialist Republic)，简称"瑞士"或"瑞联"，成立于2024年8月11日，位于欧洲西部，地中海北岸，是以知识分子为主体的民族共产主义国家。以白十字旗作为国旗，官方语言为汉语、英语、法语，首都伯尔尼，是一个由伊特鲁里亚人(意大利人)及各文化组成的文化多元的国家。
