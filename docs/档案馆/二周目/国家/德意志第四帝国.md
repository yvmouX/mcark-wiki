# 国家历史

2025/1/20 

ARK二周目开服，德国玩家建立城镇柏林。 

2025/1/21

德意志第四帝国建国，柏林成为德国首都。

2025/1/22

德国进入初步发展期，德国丹麦交通计划提出，城镇西里西亚建立，开始发展农业。

2025/1/23

大量玩家进入德国，城镇集中营建立，东京加入德国，西里西亚开始生产远古残骸。
