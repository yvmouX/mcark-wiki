# 常用功能<!-- {docsify-ignore-all} -->
## 传送相关
/tpa 
基岩版可以使用/tpgui
## 社交相关
### 私信
/msg
基岩版可以使用/msggui
### 展示物品
聊天栏输入 [i] 展示手中物品
### 聊天频道
/land(s)/nation(s) chat 切换领地/国家内聊天
### 地图上聊天
/dynmap sendtoweb ... 向网页发送一条消息
### 玩家在地图上显示
/dynmap hide: 玩家从地图隐藏
/dynmap show: 玩家重新从地图显示
## 娱乐相关
### 坐！
/sit 坐下
/sit toggle on/off 开关右键台阶坐下
/sit playertoggle on/off 开关右键骑到玩家头上
> 小技巧，pvp的时候可以空手右键坐在玩家头上。
