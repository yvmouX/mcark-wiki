# 空袭事件

## _通过特定的空袭指引器召唤空袭_

---

**标*表示对方块无破坏效果**


### 轰炸

对大面积区域投放炸弹，实现对区域的火力覆盖

### 精准打击-I

向指定地点投下大当量炸弹，对目标区域造成毁灭性打击

### 精准打击-II

向指定地点投下更大当量的炸弹，对目标区域造成毁灭性打击

### 燃烧弹*

对区域投放大量燃烧弹
